# guide-web-site

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自包含特定的内容或功能，彼此之间没有强关联，也不属于某个统一网站或域名体系。

## 项目简介

guide-web-site 是一个以 HTML 页面为单位的归档仓库。你可以在这里找到一系列独立、自包含的 HTML 文件，它们可能用于展示信息、提供工具界面或作为示例页面。仓库本身不追求流量或排名，仅作为结构化存放与版本管理的场所。

## 目录说明

仓库根目录下直接存放 HTML 页面文件。每个页面通常以简洁、有意义的文件名命名。如果需要引用公共资源（如 CSS、JavaScript 或图片），会在页面所在位置附近创建配套目录，但原则上每个页面尽量自包含，减少外部依赖。

- `*.html` — 独立 HTML 页面
- `assets/`（可选）— 部分页面引用的静态资源文件夹
- `README.md` — 本文件

## 页面归档说明

每个 HTML 页面均经过基础验证，确保在主流浏览器中可正常渲染。归档时尽量保留页面的原始结构与样式，仅对明显错误进行修正。页面内容可能随时间更新，但历史版本可通过 Git 提交记录追溯。

## 维护说明

- 本仓库不定期更新，新增页面会以独立提交的形式加入。
- 不保证所有页面的长期兼容性或链接有效性（页面内若存在外部引用，可能因外部资源变动而失效）。
- 欢迎通过 Issue 提出关于页面内容或结构的建议，但请注意本仓库聚焦于归档功能，不接受商业化合作或推广请求。
- 如需引用本仓库中的页面，请遵循开源精神，保留页面内原有的作者或版权信息（如有）。

## 许可

除非页面内另有明确声明，本仓库默认采用 [MIT License](LICENSE)。具体页面可能包含额外的使用条款，请以页面自身标注为准。
